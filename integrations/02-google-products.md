# Google Products Integration — Complete Field Guide\n\n> Author: Shrinivas Ramaprasad | April 2026  \n> Credential setup: `credentials/01-credentials-part1.md`\n\n---\n\n## Google Sheets — Every Field\n\n**Credential:** Google Sheets OAuth2 (same as Google OAuth2)  \n**Spreadsheet ID:** From URL: `docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit`\n\n### Operation: Read Rows\n| Field | Sample Value | What It Does | If Wrong |\n|---|---|---|---|\n| Spreadsheet | `https://docs.google.com/spreadsheets/d/1BxiM...` | Which spreadsheet | 404 not found |\n| Sheet | `Sheet1` | Which tab | Sheet not found |\n| Filters → Column | `Email` or `A` | Filter by this column | Returns all rows |\n| Filters → Value | `active` | Match this value | Returns wrong rows |\n| Range | `A1:F200` | Limit to range | Default: entire sheet |\n| Return All | ON | Get all matches | OFF: first match only |\n\nOutput: `{ \"Name\": \"Shrinivas\", \"Email\": \"s@x.com\", \"Status\": \"active\" }`  \nAccess: `{{ $json.Email }}` `{{ $json.Name }}`\n\n### Operation: Append Row\n| Field | Sample Value | What It Does |\n|---|---|---|\n| Spreadsheet | URL | Target spreadsheet |\n| Sheet | `Leads` | Target tab |\n| Column → Name | `Email` | Maps to this header column |\n| Column → Value | `{{ $json.email }}` | Data to write |\n| Value Input Option | `USER_ENTERED` | Interprets formulas and date formats |\n\n**Sample column mappings:**\n```\nName:        {{ $json.name }}\nEmail:       {{ $json.email }}\nCompany:     {{ $json.company }}\nSource:      Webhook\nCreated At:  {{ $now.toFormat('dd/MM/yyyy HH:mm') }}\nStatus:      New\n```\n\n### Operation: Append or Update Row\n| Field | Sample | What It Does |\n|---|---|---|\n| Match Column | `Email` | Find existing row by this column |\n| Match Value | `{{ $json.email }}` | Match value |\n| Columns | key-value pairs | Fields to set |\n\nBehavior: if row with matching email exists → update it; else → append new row.\n\n**✅ Best practices:**\n- Always use header row 1 with clear column names\n- Use `Append or Update` with email as match key for CRM-style data\n- Use `USER_ENTERED` for dates and formulas\n\n**⚠️ Common mistakes:**\n| Mistake | Effect | Fix |\n|---|---|---|\n| Sheet name wrong case | Sheet not found | Names are case-sensitive: Sheet1 ≠ sheet1 |\n| Column name mismatch | Data goes wrong column | Header names must EXACTLY match |\n| Wrong spreadsheet URL | 404 error | Copy URL from browser address bar |\n\n---\n\n## Gmail — Every Field\n\n### Operation: Send Email\n| Field | Type | Sample | Required | Notes |\n|---|---|---|---|---|\n| From Email | Email | `shrinivas@example.com` | Yes | Must match authorized Gmail account |\n| From Name | Text | `Shrinivas Ramaprasad` | No | Display name in recipient's inbox |\n| To | Email | `{{ $json.recipientEmail }}` | Yes | Comma-separate multiple |\n| CC | Email | `manager@company.com` | No | Carbon copy |\n| BCC | Email | `archive@company.com` | No | Blind carbon copy |\n| Subject | Text | `Ticket #{{ $json.id }} Updated` | Yes | Empty subject looks like spam |\n| Email Format | HTML/Text | `HTML` | Yes | HTML for rich formatting |\n| Message | HTML | `<p>Dear {{ $json.name }},</p>...` | Yes | Supports full HTML |\n| Attachments | Binary field | `data` | No | Binary from previous node |\n\n**Sample HTML email:**\n```html\n<div style=\"font-family:Arial;max-width:600px\">\n  <h2>IT Ticket Updated</h2>\n  <p>Dear {{ $json.name }},</p>\n  <p>Ticket <b>#{{ $json.ticketId }}</b> status: <b>{{ $json.status }}</b></p>\n  <p>Updated: {{ $now.toFormat('dd MMM yyyy HH:mm') }}</p>\n  <p><a href=\"{{ $json.ticketUrl }}\">View Ticket</a></p>\n</div>\n```\n\n### Operation: Get Many (Read Emails)\n| Field | Sample | What It Does |\n|---|---|---|\n| Return All | ON | Fetch all matching emails |\n| Limit | `50` | Max to return |\n| Filters → Read Status | `Unread` | Unread only / Read only / Any |\n| Filters → Labels | `INBOX` | Filter by label |\n| Filters → Sender | `noreply@stripe.com` | From specific sender |\n| Filters → Subject | `Invoice` | Subject contains |\n| Filters → Received After | `{{
