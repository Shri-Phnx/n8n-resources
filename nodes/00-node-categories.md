Language | Purpose |\n|---|---|---|\n| Code | JavaScript or Python | Any custom logic |\n| AI Transform | Natural language | Simple AI transformations |\n| LangChain Code | JavaScript | Custom LangChain pipelines |\n\n### Code Node Modes\n| Mode | Use | Access |\n|---|---|---|\n| Run Once for All Items | Aggregate, group, loop manually | `$input.all()` |\n| Run Once for Each Item | Transform per item | `$json` |\n\n### JavaScript Built-in Variables\n| Variable | Returns |\n|---|---|\n| `$json` | Current item data |\n| `$input.all()` | All items as array |\n| `$input.first()` | First item |\n| `$node['Name'].json` | Another node's output |\n| `$now` | Current time (Luxon) |\n| `DateTime` | Luxon date library |\n| `$vars.MY_VAR` | n8n Variable value |\n\n### Required Return Format\n```javascript\n// Always: array of objects with 'json' key\nreturn [{ json: { field: 'value', count: 42 } }];\n// Multiple items:\nreturn items.map(i => ({ json: { ...i.json, processed: true } }));\n```\n\n**Common mistakes:**\n| Wrong | Fix |\n|---|---|\n| `return { name: 'x' }` | `return [{ json: { name: 'x' } }]` |\n| `$json.user.name` when null | `$json.user?.name ?? 'Unknown'` |\n| Not awaiting async calls | `await $helpers.request({...})` |\n\n---\n\n## Category 4 — Webhook Nodes\n\n> Full details: `02-webhook-deep-dive.md`\n\n| Node | Direction | Purpose
